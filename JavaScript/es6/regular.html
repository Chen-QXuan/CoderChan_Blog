    正则扩展

    # 正则扩展

    正则其实是一个非常难懂的知识点，要是有人能完全掌握，那真的是非常厉害，在这里就简单的说下
    首先分为两种风格：JS 风格 和 perl 风格

    # JS风格

    通过构造函数 new RegExp("正则表达式") new RegExp()接受两个参数，new RegExp( pattern, modifiers );

    • pattern：匹配模式，可以是字符串或者RegExp对象；
    • modifiers：匹配特征(修饰符)，描述了检索是否是全局，区分大小写等；

    i - 修饰符是用来执行不区分大小写的匹配
    g - 修饰符是用于执行全文的搜索（而不是在找到第一个就停止查找,而是找到所有的匹配）

    let reg1 = new RegExp("a");  //查找一个字符串内是否含有a
let re = new RegExp('a', 'i');  //第一个是查找的对象，第二个是选项    
console.log( reg1.test("abc123") );  // true
console.log( reg1.test("a") );  // true
console.log( reg1.test("12a3") );  // true
console.log( reg1.test("qwe") );  // false

    # perl风格

    / pattern /modifiers ,且可以跟多个，不分顺序

    var reg1 = /abc/;  // 字符串中含有abc这个整体
var reg1 = /abc/i;  // 字符串中含有abc这个整体，不区分大小写    

console.log( reg1.test("abc123") );// true
console.log( reg1.test("a123bc") );// false
console.log( reg1.test("123abc") );// true

    # 在JavaScript中的应用

    # 1、正则表达式RegExp对象本身的方法

    • # test( )

    只能测试是否包含，返回一个布尔类型
    语法： RegExpObject.test(string)。

    var reg = /abc/;
console.log( reg.test("abc123") )   // true

    • # exec( )

    可以实现匹配全局，并返回分组的结果
    语法： RegExpObject.exec(string)

    let str = 'Hello World'
let regExp = /o/g     // 若结尾未加g，则每次输出都是第一个结果。    
console.log(regExp.exec(str))
// [ 'o', index: 4, input: 'Hello World', groups: undefined ]         
console.log(regExp.exec(str))
// [ 'o', index: 7, input: 'Hello World', groups: undefined ]      
console.log(regExp.exec(str))  // null

    # 2、支持正则表达式的 String 对象的方法

    • # match( )

    会返回所匹配到的内容。以数组形式返回
    语法： string.match(regexp)

    let str = 'Hello World'
let regExp = /o/   
console.log(str.match(regExp))
// ['o', index: 4, input: 'Hello World', groups: undefined]    
let regExp = /o/g
console.log(str.match(regExp))  //['o','o']

    • # replace( )

    第一个参数是正则表达式，代表匹配的内容，第二个参数是替换的字符串或者一个回调函数
    语法：string.replace(regexp,newvalue)

    let str = 'Hello World'
let regExp = /o/g   
console.log(str.replace(regExp,"z"))   // Hellz Wzrld

    • # search( )

    返回第一个匹配字符串的索引，如果没有匹配到，则返回-1
    语法：string.search(searchvalue)

    let str = 'Hello World'
let regExp = /o/g   
console.log(str.search(regExp))   // 4

    # 边界符

    • ^ 以什么开头
    var reg = /^a/;  // 以a开头   
console.log( reg.test("a") );  // true
console.log( reg.test("a123") );  // true
console.log( reg.test("123a") );  // false
console.log( reg.test("12a3") );  // false
    • $ 以什么结尾
    var reg = /abc$/;  // 以abc结尾   
console.log( reg.test("abc") );  // true
console.log( reg.test("123abc") );  // true
console.log( reg.test("abc123") );  // false
console.log( reg.test("ab123c") );  // false
    • ^和$一起使用表示,精确匹配 精确匹配就是一定是要这个字符串
    var reg = /^abc$/;  // 精确匹配  只能是这个字符串   
console.log( reg.test("abc") );  // true
console.log( reg.test("abc123") );  // false
console.log( reg.test("abc123abc") );  // false

    # 字符类

    字符类表示有一系列字符可供选择，只要匹配其中一个就可以了。所有可供选择的字符都放在方括号[ ]内。

    • [ ] 方括号,表示有一系列字符可供选择，只要匹配其中一个就可以了
    var reg = /[abc]/;  // 表示含有a或者b或者c其中一个就可以了   
console.log( reg.test("a") );  // true
console.log( reg.test("b") );  // true
console.log( reg.test("c") );  // true
console.log( reg.test("a123") );  // true
console.log( reg.test("1b23") );  // true
console.log( reg.test("123c") );  // true
console.log( reg.test("abc") );  // true
console.log( reg.test("456") );  // false
    • - 范围符

    [-] 方括号内部加上 - 表示范围，比如[a-z]表示a到z的26个英文字母任何一个字母

    var reg = /[0-9]/;  // 表示含有0-9其中一个数字,包括0和9   
console.log( reg.test("a") );  // false
console.log( reg.test("a123") );  // true
console.log( reg.test("1b23") );  // true
console.log( reg.test("0") );  // true
console.log( reg.test("9") );  // true
    • 字符组合
    var reg = /[3-5a-c]/;// 含有3~5之间,a~c之间任意一个数
console.log( reg.test("3") );  // true
console.log( reg.test("4") );  // true
console.log( reg.test("5") );  // true
console.log( reg.test("6") );  // false
console.log( reg.test("a") );  // true
console.log( reg.test("b") );  // true
console.log( reg.test("c") );  // true
console.log( reg.test("d") );  // false
console.log( reg.test("qwe3") );  // true   
var reg2 = /^[3-5a-c]$/;  // 精确匹配 3~5并且是a~c其中一个
console.log( reg2.test(3) );  // true
console.log( reg2.test(4) );  // true
console.log( reg2.test(5) );  // true
console.log( reg2.test(6) );  // false
console.log( reg2.test("a") );  // true
console.log( reg2.test("b") );  // true
console.log( reg2.test("c") );  // true
console.log( reg2.test("d") );  // false
console.log( reg2.test("3a") );  // false
console.log( reg2.test("c5") );  // false
    • ^ 取反符

    方括号内部加上 ^ 表示取反(必须加在方括号内才是取反，否则是边界符)。只要包含方括号内的字符，都返回 false。也就是 查找到任何不在方括号内的内容返回 true

    var reg = /[^abc]/;  // 含有 不是a或者b或者c其中一个 返回true    
console.log( reg.test("a") );  // false
console.log( reg.test("b") );  // false
console.log( reg.test("c") );  // false
console.log( reg.test("d") );  // true
console.log( reg.test("a123") );  // true
console.log( reg.test("abc") );  // false
console.log( reg.test("123") );  // true
console.log( reg.test("a1") );  // true

    # 量词符

    量词符用来设定某个模式出现的次数。
    默认某个模式出现的次数都是一次 比如/a/ 含有一个a
    量词符是加到字符的后面,修饰前面的字符出现多少次

    • * 重复0次或更多次
    var reg = /a*/;   
console.log( reg.test("a") );  // true
console.log( reg.test("123") );  // true
console.log( reg.test("aaa") );  // true
console.log( reg.test("abc") );  // true
    • + 重复1次或更多次 至少 1 次
    var reg = /a+/;
console.log( reg.test("a") );  // true
console.log( reg.test("123") );  // false
console.log( reg.test("aaa") );  // true
console.log( reg.test("abc") );  // true   
var reg4 = /^a+$/;  // 精确匹配 a至少出现一次
console.log( reg4.test("a") );  // true
console.log( reg4.test("123") );  // false
console.log( reg4.test("aaa") );  // true
console.log( reg4.test("abc") );  // false
console.log( reg4.test("bca") );  // false
    • ？ 重复0次或1次
    var reg = /a?/;  // 精确匹配 a出现0次或者1次   
console.log( reg.test("a") );  // true
console.log( reg.test("123") );  // true
console.log( reg.test("abc") );  // true
    • {n} 重复n次
    var reg = /a{2}/;  // 出现 2次连续a 就符合要求
console.log( reg.test("aa") );  // true
console.log( reg.test("a") );  // false
console.log( reg.test("aaa") );  // true
console.log( reg.test( 123 ) );  // false
console.log( reg.test( "a123a" ) );  // false
console.log( reg.test( "a123aa" ) );  // true   
var reg2 = /^a{2}$/;  // 精确匹配 a出现2次
console.log( reg2.test("aa") );  // true
console.log( reg2.test("a") );  // false
console.log( reg2.test("aaa") );  // false
    • {n,} 重复n次或更多次
    var reg = /a{3,}/; // 出现连续3次a 或者更多次连续的a
console.log( reg.test("a") );  // false
console.log( reg.test("aa") );  // false
console.log( reg.test("aaa") );  // true
console.log( reg.test("aa123a") );  // false
console.log( reg.test("aaa123a") );  // true
console.log( reg.test("aaaaa") );  // true   
var reg2 = /^a{3,}$/;  // 精确匹配 出现连续3次a 或者更多次连续的a
console.log( reg2.test("a") );  // false
console.log( reg2.test("aa") );  // false
console.log( reg2.test("aaa") );  // true
console.log( reg2.test("aaa123a") );  // false
console.log( reg2.test("aaaaa") );  // true
    • {n,m} 重复n到m次
    var reg = /^a{3,5}$/;  // 精确匹配  连续3次~5次a   
console.log( reg.test("a") );  // false
console.log( reg.test("aa") );  // false
console.log( reg.test("aaa") );  // true
console.log( reg.test("aaa123a") );  // false
console.log( reg.test("aaaaa") );  // true
console.log( reg.test("aaaaaa") );  // false

    # 括号总结

    • 大括号{ } 量词符，里面表示重复次数
    • 中括号[ ] 字符集合，匹配方括号中的任意字符
    • 小括号( ) 表示优先级，让它们成为一组
