    资源事件

    # 资源事件

    • # load —— 页面载入完毕后执行

    window.onload = function ( ) {}

    • # beforeunload —— 在即将离开当前页面（刷新或关闭）执行

    window.onbeforeunload = function (event) {}

    # 视图事件

    • # document.documentElement.scrollTop

    获取或设置文档元素卷起来的部分（文档元素的内容从顶部滚动出去的部分）

    • # resize —— 监听浏览器或窗口大小改变时执行

    window.onresize = function ( ) {}

    • # scroll —— 监听窗口发生滚动时执行

    window.onscroll = function ( ) {}

    # event 对象

    所有的事件处理函数，都有一个固定参数，这个参数就是事件对象。

      let div = document.querySelector('div')
  div.onclick = (event) => {
    console.log(event); 
    // PointerEvent {isTrusted: true, pointerId: 1, width: 1, height: 1, pressure: 0, …}
  }

    • # event.target —— 事件源

    谁真正触发了这个 onclick 方法，事件的源头就是谁

    • # event.type —— 事件类型

    • # event.currentTarget —— 其事件处理程序当前正在处理事件的那个元素（相当于 this）

    • # event.preventDefault( ); —— 阻止浏览器默认行为

    • # event.stopPropagation( ); —— 阻止事件冒泡或捕获

    • # 按键的键码

      • event.key
      • event.code
      • event.keyCode
      • event.altKey —— 返回当前事件触发时，“ALT”键是否被按下
      • event.ctrlKey —— 返回当前事件触发时，“Ctrl”键是否被按下
      • event.shiftKey —— 返回当前事件触发时，“Shift”键是否被按下

    • # 该参数关于鼠标

      • event.button —— 当前事件被触发时,鼠标哪个按钮被点击。（鼠标左键0; 鼠标中键是1; 鼠标右键是2; 浏览器后退按钮是3; 浏览器的前进按钮是4）
      • event.clientX / event.clientY —— 鼠标指针相对于浏览器可视区域的坐标
      • event.screenX / event.screenY —— 鼠标指针相对于电脑屏幕的坐标
      • event.pageX / event.pageY —— 鼠标相对于文档的位置（包括滚动条的距离）
      • event.offsetX / event.offsetY —— 返回发生事件的地点在 事件源元素的坐标系统中 的x坐标和y坐标（从内边距计算, 边框为负数）

    # 鼠标事件

    • # click —— 点击事件

    • # dblclick —— 双击事件

    • # mouseover —— 鼠标移入元素上方（鼠标进入）

    • # mouseout —— 鼠标移出元素

    • # mousedown —— 鼠标按下

    • # mouseup —— 鼠标松开

    • # mousemove —— 鼠标移动

    • # mouseenter —— 鼠标进入（鼠标移入）

    • # mouseleave —— 鼠标离开（鼠标移出）

    • # contextmenu —— 鼠标右键展开菜单

    # 按键事件

    • # keydown —— 按键按下 （按住不放会一直触发）

    • # keyup —— 按键松开

    • # keypress —— 当一个键被按下并且该键 产生一个字符值 时会触发该事件（按住不放会一直触发）

    # 表单事件

    • # reset —— form表单重置时 触发

    • # submit —— form表单提交时 触发

    • # focus —— 获取焦点（按钮, 单选框, 复选框 可以使用）

    • # blur —— 获取焦点（按钮, 单选框, 复选框 可以使用）

    • # input —— 输入框的值发生改变了, 就会触发input方法（不管输入还是删除）

    • # change —— 域内容发生改变（改变了值并且失去焦点以后才会触发）

    # 关于宽高

    • # offset 系列（翻译 偏移）

      • 元素.offsetWidth —— 元素的宽度（占据的空间位置的宽度）: width + padding + border
      • 元素.offsetHeight —— 元素的高度（占据的空间位置的宽度）: height + padding + border
      • 元素.offsetTop / 元素.offsetLeft —— 位置：代表的离自己最近已经定位的父元素的 left 和 top 值，如果父元素没有定位，则向上继续查找，直到找到 body

    • # client 系列（可视区）

      • 元素.clientWidth —— 元素可视区的宽度：width + padding
      • 元素.clientHeight —— 元素可视区的高：height + padding
      • 元素.clientLeft —— 左边框的宽度
      • 元素.clientTop —— 上边框的宽度
      • document.documentElement.clientWidth —— 可视区的宽度
      • document.documentElement.clientHeight —— 可视区的高度

    • # scroll 系列（滚动。如果没有滚动条 scroll 和 client 是一样的）

      • 元素.scrollWidth —— 元素的宽度(包含滚动的部分) 如果没有滚动条，则是可视区的宽
      • 元素.scrollHeight —— 元素的高度(包含滚动的部分) 实际内容的部分（超出盒子）。如果没有滚动条，则是可视区高度 height + padding
      • 元素.scrollTop —— 垂直滚动出去的部分
      • 元素.scrollLeft —— 水平滚动出去的部分
