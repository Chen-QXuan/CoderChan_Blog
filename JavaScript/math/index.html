    # Math

    Math 是一个内置对象，它拥有一些数学常数属性和数学函数方法。Math 不是一个函数对象。
    Math 用于 Number 类型。它不支持 BigInt。
    与其他全局对象不同的是，Math 不是一个构造器。Math 的所有属性与方法都是静态的。

    以下将罗列出常用的方法，更多属性方法可参考 MDN-Math (opens new window)。

    • # Math.abs( ) —— 取绝对值

    Math.abs(-Infinity);  // Infinity
Math.abs(Infinity);  // Infinity
Math.abs(-1);  // 1
Math.abs(-0);  // 0
Math.abs(1);  // 1

// Math.abs() 将其参数强制转换为数字。无法强制转换的值将变成 NaN
Math.abs("-1");  // 1
Math.abs("");  // 0
Math.abs([]);  // 0
Math.abs([2]);  // 2
Math.abs([1, 2]);  // NaN
Math.abs({});  // NaN
Math.abs("string");  // NaN
Math.abs();  // NaN

    • # Math.pow( ) —— 求n次方

    语法： Math.pow( 底数，幂 )

    Math.pow(2,4);  // 16

// 幂运算符 **  ——— ES7
console.log(2 ** 4);  // 16

    • # Math.sqrt( ) —— 求平方根

    Math.sqrt(9); // 3
Math.sqrt(1);  // 1

// 如果参数为负值，则 sqrt 返回NaN 
Math.sqrt(-1); // NaN
Math.sqrt(-0); // -0

    • # Math.cbrt( ) —— 求立方根

    Math.cbrt(125);  // 5

Math.cbrt(NaN);  // NaN
Math.cbrt(-1);  // -1
Math.cbrt(-0);  // -0
Math.cbrt(-Infinity);  // -Infinity
Math.cbrt(0);  // 0
Math.cbrt(1);  // 1
Math.cbrt(Infinity);  // Infinity
Math.cbrt(null);  // 0

    • # Math.ceil( ) —— 向上取整

    Math.ceil(1.0002);  // 2
Math.ceil(-6.002);  // -6

Math.ceil(-0);  // -0
Math.ceil(0);  // 0
Math.ceil(Infinity);  // Infinity

    • # Math.floor( ) —— 向下取整

    Math.floor(3.999);  // 3

Math.floor(-45.05);  // -46
Math.floor(-0);  // -0
Math.floor(0);  // 0

    • # Math.max( ) —— 求最大值

    Math.max(1, 3, 2, 6, 5);  // 6
Math.max(-10, -20);  // -10

    • # Math.min( ) —— 求最小值

    Math.min(1, 3, 2, 6, 5);  // 1
Math.min(-10, -20);  // -20

    • # Math.random( ) —— 求随机数

    Math.random();  // 返回 0-1 之间的随机数，即 [0, 1) 。（0取的到，1取不到）

Math.random() * n;  // 返回 0-n 之间的随机数,即 [0, n)。（0取的到，n取不到）

// 求两数之间的随机数
Math.random() * (max - min) + min;  // 返回 min 至 max 之间的随机数。（min 取的到，max 取不到）
Math.random() * (100 - 60) + 60;   // 返回 [60, 100) 的数值
//  或
Math.random() * n + x;  // 返回 x 至 x+n 之间的随机数。（x 取的到，x + n 取不到）
Math.random() * 20 + 10;  // 返回 [10, 30) 的数值

    • # Math.round( ) —— 四舍五入保留整数

    Math.round(20.49);  // 20
Math.round(20.5);  // 21

Math.round(-20.5);   // -20
Math.round(-20.51);  // -21

    • # Math.trunc( ) —— 只保留整数部分

    Math.trunc(13.37)    // 13
Math.trunc(42.84)    // 42
Math.trunc(0.123)    //  0
Math.trunc(-0.123)   // -0
Math.trunc("-1.123") // -1
Math.trunc(NaN)      // NaN
Math.trunc("foo")    // NaN
Math.trunc()         // NaN

    • # Math.round( ) —— 求一个数字的符号

    此函数共有 5 种返回值，分别是 1, -1, 0, -0, NaN. 代表的各是正数，负数，正零，负零，NaN。

    Math.sign(3);      //  1
Math.sign(-3);     // -1
Math.sign("-3");   // -1
Math.sign(0);      //  0
Math.sign(-0);     // -0
Math.sign(NaN);    // NaN
Math.sign("foo");  // NaN
Math.sign();       // NaN
