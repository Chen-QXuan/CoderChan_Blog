    Object API

    # Object API

    • # hasOwnProperty( )

    建议使用 Object.hasOwn( ) 替代此方法

    作用： 判断对象自身属性中是否具有指定的属性
    语法： obj.hasOwnProperty(prop)
    返回值： 若具有指定属性则返回 true, 否则返回 false 参数：
    prop：（必需）要测试属性的 String 类型的名称或者 Symbol

    示例：

    const obj = {}
obj.hasOwnProperty('prop');  // false
obj.prop = 'exists';
obj.hasOwnProperty('prop');  // true
delete obj.prop;
obj.hasOwnProperty('prop');  // false

'toString' in obj  // true
obj.hasOwnProperty('toString');  // false

    • # Object.hasOwn( )

    Object.hasOwn( ) 旨在取代 Object.hasOwnProperty( )。因为它适用于使用 Object.create(null) 创建的对象以及覆盖了继承的 hasOwnProperty() 方法的对象。

    作用： 判断对象自身属性中是否具有指定的属性
    语法： Object.hasOwn(obj, prop)
    返回值： 如果指定的对象中直接定义了指定的属性，则返回 true，否则返回 false。如果属性是继承的 则返回 false
    参数：
    obj：（必需）要检测的对象 prop：（必需）要测试属性的 String 类型的名称或者 Symbol

    示例：

    const obj = { name: 'coderChan' }
Object.hasOwn(obj, 'name');  // true

'toString' in obj  // true
Object.hasOwn(example, 'toString');  // false

    • # Object.keys( )

    作用： 返回一个由给定对象的所有可枚举 属性 的字符串数组（即 自身属性名组成的数组）
    语法： Object.keys(obj)
    参数：
    obj：（必需）要返回其枚举自身 属性 的对象

    注意

    在 ES5 里，如果此方法的参数不是对象（而是一个原始值），那么它会抛出 TypeError。
    在 ES6 中，非对象的参数将被强制转换为一个对象。

    示例：

    const obj = {name: 'coderChan', height: 180};
Object.keys(obj);  // ['name', 'height']

// 字符串、数组 返回的是字符串下标
const arr = ['a', 'b', 'c'];
Object.keys(arr);  // ['0', '1', '2']
const str = 'Coder';
Object.keys(str);  // ['0', '1', '2', '3', '4']

// 数字类型返回空字符串
const num = 1234;
Object.keys(num);  // []

    • # Object.values( )

    作用： 返回一个对象自身的所有可枚举 属性值 的数组
    语法： Object.values(obj)
    参数：
    obj：（必需）要返回可枚举 属性值 的对象

    示例：

    const obj = { name: 'coderChan', height: 180 };
Object.values(obj);  // ['coderChan', 180]

const arr = ['a', 'b', 'c'];
Object.values(arr);  // ['a', 'b', 'c']

const str = 'Coder';
Object.values(str);  // ['C', 'o', 'd', 'e', 'r']

// 数字类型返回空字符串
const num = 1234;
Object.values(num);  // []

    • # Object.entries( )

    作用： 返回一个对象自身的所有可枚举属性的 键值对 数组
    语法： Object.entries(obj)
    参数：
    obj：（必需）要返回可枚举属性的 键值对 的对象

    示例：

    const obj = { name: 'coderChan', height: 180 };
Object.entries(obj);  // [['name', 'coderChan'], ['height', 180]]

const arr = ['a', 'b', 'c'];
Object.entries(arr);  // [['0', 'a'], ['1', 'b'], ['2, 'c']]

const str = 'Ent';
Object.entries(str);  // [['0', 'E'], ['1', 'n'], ['2, 't']]

// 数字类型返回空字符串
const num = 1234;
Object.entries(num);  // []

    • # Object.fromEntries( )

    作用： 此方法是 Object.entries() 的逆操作，用于将一个键值对数组转为对象
    语法： Object.fromEntries(iterable)
    参数：
    iterable：（必需）类似 Array 、 Map 或者其它实现了可迭代协议 (opens new window)的可迭代对象

    示例：

    Object.fromEntries([
  ['foo', 'bar'],
  ['baz', 42]
]);  // { foo: "bar", baz: 42 }

// 该方法的主要目的，是将键值对的数据结构还原为对象，因此特别适合将 Map 结构转为对象
const map = new Map([
  ['foo', 'bar'],
  ['baz', 42]
]);

Object.fromEntries(map);  // { foo: "bar", baz: 42 }

const map2 = newMap().set('foo', true).set('bar', false);
Object.fromEntries(map2);  // { foo: true, bar: false }

    • # Object.assign( )

    作用： 用于对象的合并，将源对象（source）的所有可枚举属性，复制到目标对象（target）。此方法将修改原目标对象
    语法： Object.assign(target, ...sources)
    返回值： 合并后的目标对象
    参数：
    target：（必需）目标对象，接收源对象属性的对象，也是修改后的返回值
    sources：（可选）源对象，包含将被合并的属性

    示例：

    // 如果目标对象与源对象有同名属性，或多个源对象有同名属性，则后面的属性会覆盖前面的属性
// 同名属性的处理方法是 替换！！！ 而不是 添加
const target = { a: 1, b: 2 };
const source1 = { b: 3 };
const source2 = { c: 4 };
Object.assign(target, source1, source2);  // {a: 1, b: 3, c: 4}
console.log(target);  // {a: 1, b: 3, c: 4}

// 如果只有一个参数，则直接返回该参数
const target2 = { a: 1, b: 2 }
Object.assign( target2 );  // { a: 1, b: 2 }

// 如果该参数不是对象，则会先转成对象，然后返回
Object.assign(2);  // Number {2}

// 由于undefined和null无法转成对象，所以如果它们作为参数，就会报错
Object.assign(undefined);  // Uncaught TypeError
Object.assign(null);  // Uncaught TypeError

// 如果非对象参数出现在源对象的位置（即非首参数），这些参数都会转成对象，如果无法转成对象，就会跳过。这意味着，如果undefined和null不在首参数，就不会报错
const obj = {a: 1};
Object.assign(obj, undefined) === obj  // true
Object.assign(obj, null) === obj  // true

    注意

    Object.assign( ) 方法实行的是浅拷贝，而不是深拷贝。也就是说，如果源对象某个属性的值是对象，那么目标对象拷贝得到的是这个对象的引用。

    const obj = { a: { b: 1 } };
const obj2 = Object.assign({}, obj);  // { a: { b: 1 } } 

obj.a.b = 2;
console.log(obj2);  // { a: { b: 2 } }

    • # Object.is( )

    ES5 比较两个值是否相等，只有两个运算符：相等运算符（==）和严格相等运算符（===）。它们都有缺点，前者会自动转换数据类型，后者的 NaN 不等于自身，以及 +0 等于 -0。JavaScript 缺乏一种运算，在所有环境中，只要两个值是一样的，它们就应该相等。

    作用： 判断两个值是否为同一个值
    语法： Object.is(value1, value2)
    返回值： 若两个参数是同一个值，则返回 true, 否则返回 false。注意：引用类型 指向的地址不同，所以只要是 引用类型 比较，必定为 false
    参数： value1：（必需）被比较的第一个值
    value2：（必需）被比较的第二个值

    示例：

    Object.is(25, 25);  // true
Object.is('foo', 'foo');  // true
Object.is(null, null);  // true
Object.is(undefined, undefined);  // true

const foo = { a: 1 };
const bar = { a: 1 };
Object.is(foo, bar);  // false
Object.is([], []);  // false

console.log( +0 === -0 );  //true
console.log( NaN === NaN );  // false
Object.is(+0, -0);  // false
Object.is(NaN, NaN);  // true
Object.is(NaN, 0/0);  // true
