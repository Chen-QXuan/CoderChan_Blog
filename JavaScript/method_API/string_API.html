    String API

    # String API

    # 查找

    • # indexOf( )

    作用： 从头部往后在字符串中查找指定字符(串)，返回第一个匹配到的字符串的下标，找不到则返回 -1
    语法： string.indexOf(value, startIndex)
    返回值： 指定字符串第一次出现的索引; 若没有找到则返回 -1
    参数：
    value：（必需）需查找的字符串值
    startIndex：（可选）字符串中开始查找的索引位置。如果 startIndex >= str.length，则该方法根本不搜索调用字符串，直接返回 -1。如果 startIndex < 0，则等同于 startIndex == 0

    示例：

    const str = "Hello world, welcome to my weblog.";  

// 如果只传递一个参数, 则从整串字符串中查找指定字符(串) 
const index = str.indexOf('world');  // 6
// 第一个参数是要查找的元素, 第二个参数是从指定的下标开始往字符串 尾部 查找元素
const index2 = str.indexOf('Hello', 3);  // -1

    • # lastIndexOf( )

    作用： 从尾部往前在字符串中查找指定字符(串)，返回第一个匹配到的字符串的下标，找不到则返回 -1
    语法： string.lastIndexOf(value, startIndex)
    返回值： 指定字符串第一次出现的索引; 若没有找到则返回 -1
    参数：
    value：（必需）需查找的字符串值
    startIndex：（可选）字符串中开始查找的索引位置。如果 startIndex >= str.length ，则会搜索整个字符串。如果 startIndex < 0 ，则等同于 startIndex == 0

    示例：

    const str = "Hello world, Hello weblog.";  

// 如果只传递一个参数, 则从整串字符串中查找指定字符(串) 
const index = str.lastIndexOf('Hello');  // 13
// 第一个参数是要查找的元素, 第二个参数是从指定的下标开始往字符串 头部 查找元素
const index2 = str.lastIndexOf('Hello', 9);  // 0

    • # includes( )

    作用： 判断当前字符串里是否包含另一字符串
    语法： string.includes(item, startIndex);
    返回值： 如果包含则返回 true，否则返回 false
    参数：
    item：（必需）所要查找的字符串
    startIndex：（可选）字符串中开始查找的索引位置（传递负数时无效，该值等于0）

    示例：

    const str = 'Hello world';

// 如果只传递一个参数, 则从整个字符串中查找 
console.log( str.includes("He") );  // true  

// 第一个参数是要查找的字符串, 第二个参数是从指定的下标开始往 尾部 进行查找
console.log( str.includes("He", 3) );  // false

    • # startsWith( )

    作用： 判断当前字符串是否以另一字符串开头
    语法： string.startsWith(value, startIndex)
    返回值： 如果字符串是以指定的值 开头，则返回 true，否则 false。
    参数：
    value：（必需）所要查找的字符串
    startIndex：（可选）字符串中开始查找的索引位置，默认值为 0

    示例：

    const str = 'Hello world';

const boolean1 = str.startsWith('He');  // true
const boolean2 = str.startsWith('he');  // false

// 传递第二参数为 3，将从字符串中下标为 3 开始，到字符串结尾为准。则为 'lo world'
const boolean3 = str.startsWith('He', 3);  // false
const boolean4 = str.startsWith('lo', 3);  // true

    • # endsWith( )

    作用： 判断当前字符串是否以另一字符串结尾
    语法： string.endsWith(value, length)
    返回值： 如果字符串是以指定的值 结尾，则返回 true，否则返回 false
    参数：
    value：（必需）所要查找的字符串
    length：（可选）设置字符串的长度。默认值为原始字符串长度 string.length

    示例：

    const str = 'Coding are the best!';

const boolean1 = str.endsWith('best!');  // true
const boolean2 = str.endsWith('best');  // false

// 传递第二参数为 10，将从字符串开头，到第 10 位截止为准。则为 'Coding are'
const boolean3 = str.endsWith('best!', 10);  // false
const boolean4 = str.endsWith('are', 10);  // true

    • # charAt( )

    作用： 返回指定下标的字符
    语法： string.charAt(index)

    示例：

    const str = 'Hello world';
const value = str.charAt(4);  // 'o'
const value2 = str[4];  // 'o'

    # 裁切

    • # split( )

    作用： 把字符串切割成 数组。此方法不会改变原字符串
    语法： string.split(separator, limit)
    返回值： 返回源字符串以分隔符切割而成的一个 数组
    参数：
    separator：（可选）字符串或正则表达式，从该参数指定的地方切割
    limit：（可选）指定返回的数组的最大长度

    示例：

    const str = "Hello world.";

// 若无传递参数，则返回的数组是一个由整个字符串组成的元素
const strAry = str.split();  // ['Hello world.']
// 以'o'为切割符
const strAry2 = str.split('o');  // ['Hell', ' w', 'rld.']
// 以空字符串为切割付
const strAry3 = str.split('');  // ['H', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd', '.']

// 切割而成的数组不超过指定长度
const strAry4 = str.split('', 5);  // ['H', 'e', 'l', 'l', 'o']

    • # slice( )

    作用： 裁切字符串中的某一部分，并返回一个新的字符串。此方法不会改变原字符串
    语法： string.slice(start, end)
    返回值： 返回裁切出来的内容，包含从 start（包括该元素） 到 end （不包括该字符）的字符 参数：
    start：（必需）开始裁切的位置下标。如果为负数，则从尾部开始截取
    end：（可选）结束裁切的位置。但不包括该下标的字符

    示例：

    const str = "Hello world.";

// 如果只有一个参数, 则作为开始裁切的下标, 裁切到数组尾部
const newStr = str.slice(6);  // 'world.'
console.log(str);  // 'Hello world.'
// 如果传递两个参数, 则从 start 下标开始裁切, 裁切到 end 下标, 下标 end 裁切不到
const newStr2 = str.slice(6, 8);  // 'wo'
// 如果传递负数参数, 则从数组的尾部往前数（或者直接使用 数组的长度+负数参数）
const newStr3 = str.slice(-5, -1);  // 'orld'

    • # substring( )

    作用： 同slice方法，但是不可以传递负数
    语法： string.substring(start, end)
    返回值： 返回裁切出来的内容，包含从 start（包括该元素） 到 end （不包括该字符）的字符
    参数：
    start：（必需）开始裁切的位置下标
    end：（可选）结束裁切的位置。但不包括该下标的字符

    示例：

    const str = "Hello world.";

// 如果只有一个参数, 则作为开始裁切的下标, 裁切到数组尾部
const newStr = str.slice(2);  // 'llo world.'
// 如果传递两个参数, 则从 start 下标开始裁切, 裁切到 end 下标, 下标 end 裁切不到
const newStr2 = str.slice(2, 8);  // 'llo wo'

    # 替换

    • # replace( )

    作用： 把一个目标字符串替换成新的字符串。仅替换第一个匹配项。该方法不会修改原字符串
    语法： string.replace(regexp | str, newValue | function)
    返回值： 返回一个被替换过的新字符串
    参数：
    regexp | str：（必需）一个将被 newValue 替换的字符串。若该参数为一个正则对象，则替换掉所匹配到的内容
    newValue | function：（必需）一个将替换的新字符串。若该参数为一个函数，该函数的返回值将替换掉第一个参数匹配到的结果

    示例：

    const str = 'Hello world, Hello vuePress.';

const newStr = str.replace('Hello', 'Hi');  // 'Hi world, Hello vuePress.'

const newStr2 = str.replace(/world/, 'javascript');  // 'Hello javascript, Hello vuePress.'

const newStr3 = str.replace('Hello', value => {
  console.log(value);  // 'Hello'    该参数为第一参数的值
  return 'Bye'
});  // Bye world, Hello vuePress.

    • # replaceAll( )

    作用： 把一个目标字符串替换成新的字符串。替换掉所有的目标字符串。该方法不会修改原字符串
    语法： string.replaceAll(regexp | str, newValue | function)
    返回值： 返回一个被替换过的新字符串
    参数：
    regexp | str：（必需）一个将被 newValue 替换的字符串。若该参数为一个正则对象，则替换掉所匹配到的内容
    newValue | function：（必需）一个将替换的新字符串。若该参数为一个函数，该函数的返回值将替换掉第一个参数匹配到的结果

    示例：

    const str = 'Hello world, Hello vuePress.';
const newStr = str.replaceAll('Hello', 'Hi');  // 'Hi world, Hi vuePress.'

    # 合并

    • # concat( )

    强烈建议使用赋值操作符（+, +=）代替 concat 方法

    作用： 连接一个或多个字符串
    语法： string.concat(value1, value2, … , valueN)
    返回值： 合并后的新字符串

    示例：

    const str = 'Hello';
const newStr = str.concat(' world');  // 'Hello world'

    # 复制

    • # repeat( )

    作用： 把一个字符串重复n次。该方法不会修改原字符串
    语法： string.repeat(count)
    返回值： 返回复制指定次数并连接在一起的字符串
    参数：
    count：（必需）设置要复制的次数。该参数不可为负数

    示例：

    const str = 'coding';
const newStr = str.repeat(0);  // ''
const newStr2 = str.repeat(2);  // 'codingcoding'

// 当参数为小数时，将会被自动转换成整数
const newStr3 = str.repeat(3.5);  // 'codingcodingcoding'

    # 转大小写

    • # toUpperCase( )

    作用： 把字母转为大写
    语法： string.toUpperCase()
    返回值： 一个新的字符串，表示转换为大写的调用字符串

    示例：

    const str = 'CoderChan';
const newStr = str.toUpperCase();  // 'CODERCHAN'

    • # toLowerCase( )

    作用： 把字母转为小写
    语法： string.toLowerCase()
    返回值： 一个新的字符串，表示转换为小写的调用字符串

    示例：

    const str = 'CODERChan';
const newStr = str.toLowerCase();  // 'coderchan'

    # 去除空格

    • # trim( )

    作用： 去除字符串的头尾空格。中间的空格不做处理。此方法不会修改原字符串
    语法： string.trim()
    返回值： 返回移除头尾空格的新字符串

    示例：

    const str = '  Coder Chan  ';
const newStr = str.trim();  // 'Coder Chan'

    • # trimStart( )

    作用： 删除字符串开头的空格字符。trimLeft() 是此方法的别名。此方法不会修改原字符串
    语法： string.trimStart()           或者           string.trimLeft()
    返回值： 返回移除开头空格的新字符串

    示例：

    const str = '  Coder Chan  ';
const newStr1 = str.trimStart();  // 'Coder Chan  '
const newStr2 = str.trimLeft();  // 'Coder Chan  '

    • # trimEnd( )

    作用： 删除字符串末尾的空格字符。trimRight() 是此方法的别名。此方法不会修改原字符串
    语法： string.trimEnd()           或者           string.trimRight()
    返回值： 返回移除末尾空格的新字符串

    示例：

    const str = '  Coder Chan  ';
const newStr1 = str.trimEnd();  // '  Coder Chan'
const newStr2 = str.trimRight();  // '  Coder Chan'

    # Unicode

    • # charCodeAt( )

    作用： 返回指定下标的字符的unicode编码
    语法： string.charCodeAt(index)

    示例：

    const str = 'Hello world';
const code = str.charCodeAt(6);  // 119

    • # fromCharCode( )

    作用： 将 Unicode 编码转为一个字符
    语法： string.fromCharCode(num1, num2, ... , numN)

    示例：

    const str = String.fromCharCode(119);  // 'w'
const str2 = String.fromCharCode(72, 69, 76, 76, 79);  // 'HELLO'
